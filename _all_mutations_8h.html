Detailed Description

Convenience header aggregating all DE mutation strategy implementations.

Author
Alex Buckley

This header provides a single include point for all Differential Evolution mutation strategy classes. Use this to include all strategies at once, or include individual strategy headers for selective use.

Included Strategies:

  • Rand1: DE/rand/1 (pure exploration)
  • Rand2: DE/rand/2 (maximum exploration)
  • Best1: DE/best/1 (pure exploitation)
  • Best2: DE/best/2 (balanced)
  • RandBest1: DE/rand-to-best/1 (tunable hybrid)

Usage:

Include all strategies:

#include "Optimizer/DifferentialEvolution/Mutation/AllMutations.h"
