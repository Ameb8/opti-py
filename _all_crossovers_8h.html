Detailed Description

Convenience header aggregating all DE crossover strategy implementations.

Author
Alex Buckley

This header provides a single include point for all Differential Evolution crossover strategy classes. Use this to include all strategies at once, or include individual strategy headers for selective use.

Included Strategies:

  • BinCrossover: Binomial (uniform, independent) crossover
  • ExpCrossover: Exponential (continuous block) crossover

Usage:

Include all strategies:

#include "Optimizer/DifferentialEvolution/Crossover/AllCrossovers.h"

#include "Optimizer/DifferentialEvolution/Crossover/BinCrossover.h"
